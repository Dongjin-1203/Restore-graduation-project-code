# Restore-graduation-project-code
---
본 리포지토리는 2021년 인천대학교 물리학 연구 프로젝트 [전영재, 지동진, 정재훈]팀의 '불균형하게 도핑된 뒤틀린 이중층 구리산화물 초전도체 이론 연구'주제의 연구 진행간 작성했던 코드를 깃허브에 복원한 것입니다. [기초 논문(High-temperature superconductivity in monolayer Bi2Sr2CaCu2O8+δ)](https://www.nature.com/articles/s41586-019-1718-x)의 에너지 그래프를 시각화 한 것이고 해당 연구를 바탕으로 2-D물질을 불균형하게 도핑했을때 초전도 특성을 유지하는지에 대해 이론적인 연구를 하였고, 복잡한 양자 수식을 Python으로 계산하고 시각화 하였습니다.
